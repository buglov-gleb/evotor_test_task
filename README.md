# Evotor Test Task
This repository contains test task for Computational Linguist position at Evotor.

### Prerequisites

```
Python  > = 3.8
Requests >= 2.24.0
```

### Installing

1. Clone and navigate to command line chatbot directory
2.  Run the program
 ```
python3 evotor_test_task.py
```
3.  Now you can play quiz game.

Have fun!
